# cypress-eslint-prettier

```
$ npm init -y
$ npm i cypress
$ npx cypress open
$ npm i -D eslint prettier eslint-config-prettier
$ curl https://raw.githubusercontent.com/tacck/base-config-files/main/cypress-eslint-prettier/download.sh | sh
```
